# Speedix WordPress テーマ インストールガイド

このガイドでは、SpeedixテーマをWordPressにインストールする手順を説明します。

## インストール前の準備

### 必要なもの

- WordPress 6.0以上がインストールされたサーバー
- FTPまたはファイルマネージャーへのアクセス
- データベースへのアクセス（phpMyAdminまたはWP-CLI）

### 推奨プラグイン

- **Contact Form 7**: お問い合わせフォーム用（必須）
- **All-in-One WP Migration**: バックアップとインポート用（オプション）

## インストール手順

### ステップ1: テーマファイルのアップロード

#### 方法A: WordPressダッシュボードから

1. `speedix-theme.tar.gz` を解凍して `speedix` フォルダを取得
2. `speedix` フォルダをZIP形式で再圧縮（`speedix.zip`）
3. WordPress管理画面にログイン
4. 外観 > テーマ > 新規追加 > テーマのアップロード
5. `speedix.zip` を選択してインストール
6. 「有効化」をクリック

#### 方法B: FTPでアップロード

1. `speedix-theme.tar.gz` を解凍
2. FTPクライアントで `/wp-content/themes/` ディレクトリに接続
3. `speedix` フォルダをアップロード
4. WordPress管理画面 > 外観 > テーマ から「Speedix」を有効化

### ステップ2: 初期データのインポート

テーマには商品とページの初期データが含まれています。

#### 方法A: phpMyAdminを使用

1. サーバーのphpMyAdminにアクセス
2. WordPressデータベースを選択
3. 「SQL」タブをクリック
4. `speedix/initial-data.sql` の内容をコピー＆ペースト
5. 「実行」をクリック

#### 方法B: WP-CLIを使用（サーバーでWP-CLIが使える場合）

```bash
cd /path/to/wordpress
wp db query < wp-content/themes/speedix/initial-data.sql
```

### ステップ3: サイトロゴの設定

1. WordPress管理画面 > 外観 > カスタマイズ
2. サイト基本情報 > ロゴを選択
3. Speedixロゴ画像をアップロード
4. 公開をクリック

### ステップ4: Contact Form 7の設定

1. プラグイン > 新規追加 > 「Contact Form 7」を検索してインストール
2. 有効化
3. お問い合わせ > 新規追加
4. 以下のフォームコードを入力:

```html
<label> お名前（必須）
    [text* your-name] </label>

<label> メールアドレス（必須）
    [email* your-email] </label>

<label> 電話番号
    [tel your-phone] </label>

<label> 会社名
    [text your-company] </label>

<label> お問い合わせ内容
    [textarea your-message] </label>

[submit "送信"]
```

5. 保存後、生成されたショートコードをコピー
6. 固定ページ > お問い合わせ を編集
7. ショートコードを本文に貼り付けて更新

### ステップ5: ページ設定の確認

以下のページが正しく作成されているか確認してください：

- ホーム（フロントページ）
- Speedixについて
- 提供内容（商品一覧）
- お問い合わせ
- 会社情報
- チェックアウト
- ご注文ありがとうございます
- プライバシーポリシー
- 特定商取引法表示

#### フロントページの設定

1. 設定 > 表示設定
2. 「ホームページの表示」を「固定ページ」に変更
3. ホームページに「ホーム」を選択
4. 変更を保存

### ステップ6: パーマリンク設定

1. 設定 > パーマリンク設定
2. 「投稿名」を選択
3. カスタム構造: `/%postname%/`
4. 変更を保存

### ステップ7: 商品の確認と編集

1. 管理画面 > 商品管理
2. 既存の商品を確認
3. 必要に応じて商品情報を編集:
   - 価格
   - 説明
   - 特徴・メリット
   - 表示順

## トラブルシューティング

### 商品が表示されない

- 商品投稿タイプが正しく登録されているか確認
- 商品の「公開状態」がチェックされているか確認
- パーマリンク設定を再保存

### カートが動作しない

- ブラウザのJavaScriptが有効になっているか確認
- ブラウザコンソールでエラーを確認
- テーマのJavaScriptファイルが正しく読み込まれているか確認

### メールが送信されない

- WordPressのメール送信機能が正しく動作しているか確認
- WP Mail SMTPなどのプラグインを使用してメール送信を設定

### スタイルが崩れる

- Tailwind CSSのCDNが読み込まれているか確認
- ブラウザのキャッシュをクリア
- テーマのCSSファイルが正しく読み込まれているか確認

## All-in-One WP Migrationを使用したインストール

既存のSpeedixサイトのバックアップがある場合：

1. All-in-One WP Migrationプラグインをインストール＆有効化
2. 管理画面 > All-in-One WP Migration > Import
3. バックアップファイルを選択してインポート
4. インポート完了後、パーマリンク設定を再保存

## 次のステップ

1. サイトの全ページを確認
2. テスト注文を実行してカート機能を確認
3. Contact Form 7のテストメールを送信
4. モバイル表示を確認
5. 商品情報を最新の内容に更新
