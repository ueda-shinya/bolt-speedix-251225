# Speedix WordPress Theme

最短・最速で成果を出すWeb制作サービス「Speedix」の公式WordPressテーマ

## 概要

このテーマは、Yis株式会社が提供するSpeedixサービスのWebサイト用カスタムテーマです。商品管理、カート機能、お問い合わせフォーム統合など、LP制作サービスに特化した機能を搭載しています。

## 特徴

- **カスタム投稿タイプ「商品」**: 商品情報を管理画面から簡単に追加・編集可能
- **LocalStorageベースのカート機能**: WooCommerceを使わない軽量なショッピングカート実装
- **Tailwind CSS統合**: モダンでレスポンシブなデザイン
- **Ajax対応のチェックアウトフォーム**: スムーズな注文処理
- **完全レスポンシブ**: モバイル・タブレット・デスクトップ対応

## 必要要件

- WordPress 6.0以上
- PHP 7.4以上
- 推奨プラグイン:
  - Contact Form 7（お問い合わせフォーム用）

## インストール方法

### 方法1: 手動インストール

1. `speedix`フォルダを `/wp-content/themes/` ディレクトリにアップロード
2. WordPress管理画面 > 外観 > テーマ から「Speedix」を有効化
3. 外観 > カスタマイズ でサイトロゴをアップロード
4. 初期データのインポート（下記参照）

### 方法2: All-in-One WP Migrationを使用

1. All-in-One WP Migrationプラグインをインストール
2. 管理画面 > All-in-One WP Migration > Import
3. 提供されたバックアップファイルをインポート

## 初期データのインポート

テーマをインストール後、商品とページの初期データをインポートします：

1. phpMyAdminまたはデータベース管理ツールにアクセス
2. WordPressデータベースを選択
3. `initial-data.sql` ファイルの内容を実行

または、WP-CLI を使用:

```bash
wp db query < initial-data.sql
```

## ページテンプレート

以下の特別なページテンプレートが用意されています：

- **Front Page** (`front-page.php`): トップページ
- **Products Archive** (`archive-product.php`): 商品一覧ページ
- **Checkout Page** (`page-checkout.php`): チェックアウトページ
- **Thank You Page** (`page-thank-you.php`): 注文完了ページ
- **Contact Page** (`page-contact.php`): お問い合わせページ

## 商品の追加方法

1. 管理画面 > 商品管理 > 新規追加
2. 商品名と説明を入力
3. 商品詳細セクションで以下を設定:
   - 価格（円）
   - スラッグ（URL用）
   - 特徴・メリット（1行に1項目）
   - 画像URL
   - 公開状態（チェックで有効化）
   - 表示順
4. 公開ボタンをクリック

## カート機能について

このテーマのカート機能は、LocalStorageを使用したシンプルな見積もり依頼システムです。

- カートデータはユーザーのブラウザに保存されます
- チェックアウト時にメールで注文情報が送信されます
- 決済処理は含まれていません（別途Stripeなどの統合が必要）

## カスタマイズ

### ロゴの変更

管理画面 > 外観 > カスタマイズ > サイト基本情報 > ロゴを選択

### 色の変更

`style.css` ファイルまたはTailwind CSSのクラスを編集してください。

### フォームのカスタマイズ

Contact Form 7プラグインをインストールし、お問い合わせページにショートコードを挿入してください。

## 注文管理

受信した注文は `wp_speedix_orders` テーブルに保存されます。

管理画面での注文一覧機能は今後のアップデートで追加予定です。
